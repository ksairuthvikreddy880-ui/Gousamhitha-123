- Use Authorization header for auth endpoints
- Send JSON in body

### JavaScript/Fetch

## 🚀 Getting Started

### Start the Server
```bash
cd backend
npm install
npm run dev
# Server running on http://localhost:4000
```

### Test It
```bash
# Health check
curl http://localhost:4000/api/health

# Get products
curl http://localhost:4000/api/products

# Create product (with validation)
curl -X POST http://localhost:4000/api/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Tomatoes",
    "category": "Vegetables",
    "price": 50,
    "stock": 100
  }'
```

## 💡 How to Add New Endpoints

1. Define Joi schema in `middleware/validators.js`
2. Create controller using `asyncHandler()`
3. Add route with validation middleware
4. Return response using helper functions

**Time needed**: ~10 minutes (see `DEVELOPERS-GUIDE.md`)
